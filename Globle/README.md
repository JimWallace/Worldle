# 🌍 Globle — a native SwiftUI globe-guessing game

A native iOS remake of [Globle](https://globle-game.com/game): every day there's a
hidden **mystery country**, and you guess countries to find it. Each guess lights up
on a real 3D globe, colored by how close it is — **red is boiling hot** (right next
door) and **blue is freezing cold** (far away). Follow the warm colors to track it down.

Built as an ad-free learning tool: **every country on Earth is here** (242 countries
and territories you can guess, 199 sovereign countries that can be the daily answer),
with flags, forgiving search, and a spinnable, zoomable globe.

| | |
|---|---|
| Platform | iOS 16+ (iPhone & iPad) |
| Built with | SwiftUI + SceneKit, no third-party dependencies |
| Map data | [Natural Earth](https://www.naturalearthdata.com/) 1:50m (public domain) |

## How to play

1. A secret country is hiding somewhere on Earth.
2. Type any country's name and tap it to guess.
3. The globe colors your guess by closeness (🔥 red = next door, ❄️ blue = far away),
   spins to show it, and your guesses are ranked hottest-first.
4. Find the mystery country in as few guesses as you can. A new one every day!

## Build & run

1. Open **`Globle.xcodeproj`** in Xcode 15 or newer.
2. Select the **Globle** scheme and an iPhone/iPad simulator, then press **⌘R**.
3. To run on a real device (recommended for your kid), select your device, then in
   **Signing & Capabilities** pick your Apple ID under *Team*. Personal (free) Apple
   IDs work fine for installing on your own devices.

You may want to change `PRODUCT_BUNDLE_IDENTIFIER` (currently `com.example.globle`) to
something like `com.yourname.globle` in the target's Build Settings.

> Heads-up: this project was authored in a Linux environment without Xcode, so it has
> not been compiled by its author. The code is conservative and idiomatic, but if the
> very first build surfaces a small issue, it'll be a quick fix. One thing to glance at:
> if the globe ever renders **upside-down**, flip the single documented line in
> `Globle/Globe/SphereMesh.swift` (`uvs.append(... v ...)` → use `1 - v`).

## How it works

- **Globe** — a custom UV sphere whose texture is painted at runtime with Core Graphics
  (equirectangular projection: oceans, all land in soft green, and guessed countries
  recolored by proximity). Tiny countries are drawn as dots so microstates stay visible.
- **Closeness** — the minimum great-circle distance between the borders of your guess and
  the mystery country, so neighbors come out hottest, just like the original.
- **Daily puzzle** — chosen deterministically from the date with a seeded shuffle, so
  everyone gets the same country each day and every sovereign country comes up before any
  repeats.

## Regenerating assets

Everything generated lives under `tools/` and is fully reproducible:

```bash
python3 tools/build_countries.py   # rebuild Globle/Resources/countries.json from Natural Earth
python3 tools/make_icon.py          # rebuild the app icon
python3 tools/make_xcodeproj.py     # rebuild Globle.xcodeproj (run after adding/removing files)
```
